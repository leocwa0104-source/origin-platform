# Origin原创者平台 - 3分钟上线运营指南

## 一、项目概述

Origin原创者平台是一个AI驱动的中小创作者版权确权社交平台，核心解决原创者「确权难、维权难、变现难」三大痛点。

## 二、上线前准备

### 1. 技术准备
- **代码仓库**：将项目代码上传到GitHub/GitLab仓库
- **域名**：准备一个已备案的域名（推荐使用阿里云/腾讯云域名）
- **服务器**：本项目使用Serverless架构，无需购买服务器

### 2. 合规准备
- **网站备案**：完成域名备案（根据工信部要求）
- **增值电信业务许可证**：如果涉及付费内容，需要办理ICP许可证
- **版权相关资质**：确保平台运营符合《著作权法》要求

### 3. 支付配置
- **微信支付商户号**：申请并配置微信支付商户号
- **支付宝商户号**：申请并配置支付宝商户号
- **分账设置**：在支付平台设置分账规则，确保资金直接到账创作者账户

### 4. AI模型配置
- **豆包API**：注册并获取豆包API密钥
- **通义千问API**：注册并获取通义千问API密钥

### 5. 区块链配置
- **蚂蚁链**：注册蚂蚁链开放平台账号
- **百度超级链**：注册百度超级链开放平台账号

## 三、部署步骤

### 1. 部署到Vercel
1. **登录Vercel**：访问 https://vercel.com 并登录/注册账号
2. **导入项目**：点击「New Project」→「Import Git Repository」
3. **选择仓库**：选择你的GitHub/GitLab仓库
4. **配置项目**：
   - Project Name: origin-platform
   - Framework Preset: React
   - Build Command: `npm run build`
   - Output Directory: `dist`
   - Environment Variables: 配置必要的API密钥
5. **部署**：点击「Deploy」按钮，等待部署完成

### 2. 域名配置
1. **Vercel域名设置**：在Vercel项目中，点击「Settings」→「Domains」
2. **添加域名**：输入你的域名（如 origin.com）
3. **DNS配置**：根据Vercel提供的DNS记录，在域名注册商处配置DNS
4. **SSL证书**：Vercel会自动为你的域名生成SSL证书

### 3. 后端配置（Supabase）
1. **登录Supabase**：访问 https://supabase.com 并登录/注册账号
2. **创建项目**：点击「New Project」创建新项目
3. **获取API密钥**：在项目设置中获取API URL和Service Key
4. **配置数据库**：根据项目需求创建必要的表结构
5. **环境变量**：将Supabase的API密钥添加到Vercel的环境变量中

### 4. 存储配置（OSS/COS）
1. **阿里云OSS**：
   - 登录阿里云控制台，创建OSS存储桶
   - 配置CORS规则和访问权限
   - 获取Access Key和Secret Key

2. **腾讯云COS**：
   - 登录腾讯云控制台，创建COS存储桶
   - 配置CORS规则和访问权限
   - 获取SecretId和SecretKey

## 五、常见问题

### 1. 部署失败怎么办？
- 检查构建命令是否正确
- 检查环境变量是否配置完整
- 检查依赖是否安装正确

### 2. 支付配置遇到问题？
- 确保商户号已完成实名认证
- 确保分账功能已开通
- 检查支付回调地址是否正确

### 3. AI模型调用失败？
- 检查API密钥是否正确
- 检查API调用频率是否超限
- 检查网络连接是否正常

### 4. 区块链存证失败？
- 检查区块链API配置是否正确
- 检查网络连接是否正常
- 检查存证数据格式是否正确
